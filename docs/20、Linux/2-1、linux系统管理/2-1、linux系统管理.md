#### ifconfig

网络接口配置

直接输入ifconfig就是该命令了

```
ifconfig
```

显示含义：

```
eth0 第⼀块⽹网卡（⽹网络接⼝）
```

你的第⼀个⽹网络接⼝口可能叫做下⾯的名字

* eno1 板载网卡
* ens33 PCI-E网卡
* enp0s3 无法获取物理理信息的 PCI-E ⽹网卡
* CentOS 7 使⽤用了了⼀致性⽹网络设备命名，以上都不不匹配则使⽤用 eth0

PS:

* 网卡名称固定之后，方便编写多主机批量控制脚本
* 板载网卡有可能不被linux识别，因为非服务器主板驱动程序和linux的兼容性并不好
